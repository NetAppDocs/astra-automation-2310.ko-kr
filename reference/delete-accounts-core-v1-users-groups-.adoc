= 사용자의 그룹 삭제
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/users/{user_id}/groups/{group_id}`#

지정된 그룹 API 리소스를 제거합니다.

*부작용*

* 그룹과 관련된 다음 리소스가 삭제됩니다.
* 그룹을 참조하는 역할 바인딩




== 매개변수

[cols="5*"]
|===
| 이름 | 유형 | 에서 | 필수 요소입니다 | 설명 


| account_id | 문자열 | 경로 | 참  a| 
포함하는 계정 리소스의 ID

* 예: {{.Account}}




| user_id | 문자열 | 경로 | 참  a| 
포함하는 사용자 리소스의 ID



| group_id | 문자열 | 경로 | 참  a| 
목록에 표시할 그룹 컬렉션의 ID

|===
