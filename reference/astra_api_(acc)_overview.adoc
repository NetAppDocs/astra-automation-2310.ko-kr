= Astra API(ACC) 개요
:allow-uri-read: 


Astra Control REST API 참조 문서에는 엔드포인트, HTTP 메서드, 입력 매개변수 및 응답을 포함한 모든 REST API 호출에 대한 세부 정보가 포함되어 있습니다. 이 완벽한 참조 문서는 API를 사용하여 자동화 애플리케이션을 개발할 때 유용합니다. Astra REST API는 특정 리소스 범주에 대해 별도로 명시된 경우를 제외하고 모든 Astra Control 리소스에 대한 CRUD(생성-읽기-업데이트-삭제) 액세스를 제공합니다. 이 참조 문서는 Astra Control REST API 23.10 릴리스에 적용됩니다.
