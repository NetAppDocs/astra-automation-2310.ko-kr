= 애플리케이션 스냅샷 삭제
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/appSnaps/{appSnap_id}`#

지정된 애플리케이션 스냅샷 API 리소스를 제거합니다.

*부작용*

* 스냅샷이 진행 중인 경우 스냅샷이 취소됩니다.
* 스냅샷이 완료되면 스냅샷 데이터가 삭제됩니다.




== 매개변수

[cols="5*"]
|===
| 이름 | 유형 | 에서 | 필수 요소입니다 | 설명 


| account_id | 문자열 | 경로 | 참  a| 
포함하는 계정 리소스의 ID

* 예: {{.Account}}




| app_id | 문자열 | 경로 | 참  a| 
포함하는 앱 리소스의 ID



| appSnap_id | 문자열 | 경로 | 참  a| 
목록에 표시할 appSnap 컬렉션의 ID

|===
