= Astra API(ACC) 후크 소스 엔드포인트
:allow-uri-read: 


Hook Source API를 사용하면 실행 훅 리소스 관리와 별도로 훅 소스를 간편하게 관리할 수 있습니다. 예를 들어, 하나의 훅 소스를 여러 실행 훅에서 참조할 수 있으므로 중복된 훅 소스를 방지하고, 여러 실행 훅 리소스에서 사용되는 경우에도 단일 API 호출로 하나의 훅 소스를 업데이트할 수 있습니다.
