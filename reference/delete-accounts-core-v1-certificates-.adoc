= 인증서 삭제
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/certificates/{certificate_id}`#

지정된 Certificate API 리소스를 제거합니다.



== 매개변수

[cols="5*"]
|===
| 이름 | 유형 | 에서 | 필수 요소입니다 | 설명 


| account_id | 문자열 | 경로 | 참  a| 
포함하는 계정 리소스의 ID

* 예: {{.Account}}




| certificate_id | 문자열 | 경로 | 참  a| 
목록에 표시할 인증서 모음의 ID

|===


== 응답

[source]
----
Status: 204, Indicates if the certificate resource was deleted.
----


== 응답

[source]
----
Status: 503, Service unavailable
----
[cols="4*"]
